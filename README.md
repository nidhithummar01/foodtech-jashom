# Admin Panel (Mock Frontend)

Internal operations admin panel built with React + TypeScript + Vite + Tailwind CSS.

## Setup

1. Install dependencies:
   - `npm install`
2. Start development server:
   - `npm run dev`
3. Build for production:
   - `npm run build`
4. Run checks:
   - `npm run lint`

## Project Notes

- This project is frontend-only and uses local mock data from `src/data/*.mock.ts`.
- No backend integration, no API calls, and no real authentication are included.
- Pages implemented:
  - `/dashboard`
  - `/users`
  - `/restaurants`
  - `/orders`
  - `/reports`
  - `/settings`
- Data pages include loading, empty, and data states.
